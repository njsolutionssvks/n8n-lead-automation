# Lead Management Automation using n8n

## Overview

This project automates lead collection and processing using n8n workflows. It captures leads via webhook, stores them in Google Sheets, and sends email notifications automatically.

---

## ⚙️ Tech Stack

* n8n (workflow automation)
* Google Sheets API
* Gmail API
* Webhooks

---

## Workflow Architecture

```
Webhook → Google Sheets → Gmail Notification
```

---

## Features

* Capture leads via API/Webhook
* Store data automatically in Google Sheets
* Send instant email notifications
* Fully automated workflow
* No manual intervention required
